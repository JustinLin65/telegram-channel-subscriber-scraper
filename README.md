# Telegram Channel Scraper

*本專案是一款基於 Telethon 開發的輕量級成員抓取工具，專為社群管理者設計，用於快速導出頻道或群組的訂閱者名單進行數據分析。*
*我在經營 10K DOG 社群期間，為了方便確認代幣空投資格而開發的自動化工具*

由於 Telegram API 的限制，一般方式無法直接獲取完整的成員列表。本工具透過 **關鍵字迭代搜尋技術 (A-Z, 0-9)**，盡可能繞過限制並最大化抓取到的成員數量。

## 核心功能

- **深度搜尋抓取**：遍歷 `a-z` 與 `0-9` 關鍵字進行窮舉搜尋，突破預設的顯示限制。
- **自動去重機制**：使用 Python 字典結構（Dict）處理資料，確保輸出的成員名單唯一不重複。
- **詳細資料導出**：支援導出 User ID、Username、姓名及電話（若對方開放權限）。
- **頻率限制保護**：內建 `asyncio.sleep` 機制，有效避免因請求過快觸發 Telegram 的 Rate Limit。
- **Excel 友善格式**：輸出為 `UTF-8-sig` 編碼的 CSV 檔案，確保在 Excel 中開啟不發生亂碼。

## 使用說明

1. **環境準備**

    請確保你的電腦已安裝 Python 3.7+。

    安裝必要的套件：
    `pip install telethon`

2. **取得憑證**

    你需要從 Telegram 官方取得 API 權限：
    - 前往 [my.telegram.org](https://my.telegram.org/) 建立 App。
    - 取得 `API_ID` 與 `API_HASH`。

3. **設定參數**

    開啟 `TG.py`，修改 `設定區` 的內容：
    ```python
    api_id = '你的_API_ID'
    api_hash = '你的_API_HASH'
    phone = '你的電話'           # 格式如 +886912345678
    channel_id = '頻道連結或ID'  # 支援連結 ([https://t.me/xxx](https://t.me/xxx)) 或 ID (-100xxx)
    ```

4. **啟動程式**

    執行抓取指令：
    `python TG.py`
    
    *首次執行時需要輸入 Telegram 發送的驗證碼進行登入。*

## ⚠️ 注意事項

- **隱私權限制**：若用戶在隱私設定中隱藏了電話，程式將顯示為 "Hidden"。
- **API 限制**：本工具採用的窮舉搜尋法雖然能抓取更多成員，但仍受限於 Telegram 伺服器端的權限開放程度。
- **帳號安全**：請避免在短時間內對大量頻道進行高頻率抓取，以保護你的 Telegram 帳號不被暫時封禁。
